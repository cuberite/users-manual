Cuberite can be configured by editing various configuration files. Below is a list of all configuration files.

settings.ini
The main configuration file, it contains server-wide configuration variables.
webadmin.ini
Allows you to tweak the web admin interface, which is available at http://localhost:8080 or http://<Server IP address>:8080 by default.
world/world.ini
This file configures world-specific aspects. This is where you choose your game mode. See {{3.3 - GameMode}}. Note that if you have multiple worlds, you'll have multiple world.ini files, each stored in <World name>/world.ini.
monsters.ini
Allows you to tweak monster behaviour.
motd.txt
The Message of the Day, which is shown to players upon joining your server.
crafting.txt, brewing.txt, furance.txt
These three files allow to tweak crafting, brewing, and furance recipes.
plugins/...
Many plugins have their own configuration files. For instance, the WorldEdit config is plugins/WorldEdit/config.cfg
items.ini
Edit item IDs. You probably shouldn't edit this file unless you know what you're doing.
Note that in all .ini files, lines starting with ; are comments, and in all .txt files, lines starting with # are comments.

Permissions

Setting up player permissions is most easily done via the web admin. You can also use the rank <playername> command from the server console.
