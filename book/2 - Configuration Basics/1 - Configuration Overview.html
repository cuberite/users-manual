Cuberite can be configured by editing various files. Below is a list of all such files:

Cuberite는 다양한 파일을 편집하여 구성 할 수 있습니다. 아래는 파일들의 목록입니다.

settings.ini
The main configuration file, which contains server-wide configuration variables.
서버 전체의 구성 변수를 포함하는 주 구성 파일입니다.
webadmin.ini
Allows you to tweak the web admin interface, which is available at http://localhost:8080 or http://<Server IP address>:8080 by default.
기본적으로 http : // localhost : 8080 또는 http : // <서버 IP 주소> : 8080 을 통해 웹 관리 인터페이스를 조정할 수 있습니다.
<World name>/world.ini
This file configures world-specific aspects. This is where you choose your game mode. See {{3.3 - GameMode}}. Note that each world has its own world.ini file, each stored in <World name>/world.ini.
이 파일은 세계의 구체적인 설정을합니다. 이 파일에서 게임 모드를 선택하세요. {{3.3 - 게임모드}}를보세요. 각 세계에는 & lt; World name & gt; / world.ini 에 각각의 world.ini 파일이 있습니다.
monsters.ini
Allows you to tweak monster behaviour.
몬스터의 행동을 바꿀 수 있는 파일입니다.
motd.txt
The Message of the Day, which is shown to players upon joining your server.
플레이어가 서버에 접속할 때 보여지는 메세지 설정 파일입니다.
crafting.txt, brewing.txt, furnace.txt
These three files allow you to tweak crafting, brewing, and furnace recipes.
plugins/...
Many plugins have their own configuration files. For instance, the WorldEdit config is plugins/WorldEdit/config.cfg
많은 플러그인들은 자체 구성 파일이 있습니다. 예를 들어 WorldEdit 구성은 plugins / WorldEdit / config.cfg 입니다.
favicon.png
This is the icon that will appear in the Minecraft server list. You may replace it with your own. The icon dimensions musts be 64x64.
이것은 Minecraft 서버 목록에 나타날 아이콘입니다. 원하는 아이콘으로 변경하셔도 되지만, 아이콘 크기는 64x64 이어야합니다.
items.ini
Edit item IDs. You probably shouldn't edit this file unless you know what you're doing.
아이템 ID를 수정하는 파일입니다. 잘 모르는 경우 건드리지 마세요.
In all .ini files, lines starting with ; are comments, and in all .txt files, lines starting with # are comments.
모든 .ini 파일에서는 ; 로 시작하는 줄은 주석이며 모든 .txt 파일에서 #로 시작하는 줄ㄹ은 주석입니다.
