By default, there are three worlds: world, world_nether, world_the_end. Each world can be tweaked by editing <World Name>/world.ini. You can use this file to edit things like the spawn point location, the game mode, and the diffculty level. See {{3 - Configuring world.ini}} for details.

기본적으로 world , world_nether , world_the_end 의 세 가지 세계가 있습니다. < World Name > /world.ini 를 편집하여 각 세계를 조정할 수 있습니다. 이 파일을 사용하여 스폰 지점 위치, 게임 모드 및 난이도를 편집 할 수 있습니다. 자세한 내용은 {{3 - world.ini 파일 구성}}을 참조하십시오.
