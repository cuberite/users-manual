Cuberite is a Free and Open Source (FOSS) Minecraft-compatible Game Server. Cuberite is designed with performance, configurability, and extensibility in mind, and also aims to accurately recreate most vanilla features. Cuberite is written in C++, and there is an extensive plugin system that allows for the user to write their own plugins with Lua. In fact, many of the built in commands are implemented by the Core plugin, which has its own GitHub repository and developer community. For more information on the plugin system and how to use it, as well as how to develop for it, please see {{2.4 - Plugins}}.

Cuberite는 Free and Open Source (FOSS) Minecraft 호환 게임 서버입니다. Cuberite는 성능, 구성 가능성 및 확장 성을 염두에두고 설계되었으며 바닐라 기능을 정확하게 재현하는 것을 목표로 합니다. Cuberite는 C ++로 작성되었으며 사용자가 Lua와 함께 자신의 플러그인을 작성할 수 있는 광범위한 플러그인 시스템이 있습니다. 사실 내장 된 많은 명령은 자체 GitHub 저장소 및 개발자 커뮤니티가 있는 코어 플러그인에 의해 구현됩니다. 플러그인 시스템과 플러그인 사용법 및 개발 방법에 대한 자세한 내용은 {{2.4 - 플러그인}}을 참조하세요.
